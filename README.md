WELCOME TO OVIM!

   The only text editor you'll ever need
  
   OVim is a sort-of vim-based text editor, written in 1 night.
   
   What sets OVim apart from other Vim-based Text editors is
   that its made ENTIRELY in python! 
   
   Features include:
   
   -VERY thorough theme customization with "Theme Builder"
   
   -Syntax highlighting for bash, python3 and json
   
   -Auto-closing brackets ([{"''"}]) 
   
   -Full git integration
   
   -Built-in terminal emulator
   
   -Multiple tabs
   
   -Split displays
   
   -Fuzzy finder
   
   -Persistent config (~/.config/ovim)
   
   -!!COMPLETELY OPEN SOURCE!! BE FREE TO FORK AND MODIFY!
